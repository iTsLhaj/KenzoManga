# KenzoManga

_KenzoManga_ is a modern web application for manga enthusiasts.
Built with _React_ and _Flask_, it offers a seamless reading experience.
Users can browse, search, and read their favorite manga titles.
The platform aims to provide a user-friendly interface and a vast manga library.
<br>
