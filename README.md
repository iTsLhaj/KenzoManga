# KenzoManga



A full-stack demo project for a **manga reading and management platform**.  
Built with **Python (backend)** and **TypeScript/HTML/CSS (frontend)**.

---

## Features

- Browse and read manga chapters  
- Full-stack architecture with frontend and backend separation  
- Simple and clean user interface

> NOTE: ⚠️ This project is **archived and no longer maintained**.
